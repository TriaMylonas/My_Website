## Δεύτερο Έτος: Επέκταση στον Κόσμο του Web:
Κατά το δεύτερο έτος της εκπαίδευσής μου, επέκτεινα τις γνώσεις μου προς την ανάπτυξη WPF εφαρμογών με C#. 
Δημιούργησα μια εφαρμογή που δυναμικά δημιουργεί barcodes από κείμενο που εισάγει ο χρήστης και τα ενσωματώνει σε ένα PDF. 
Αυτή η εμπειρία με έκανε να κατανοήσω την ισχύ και την ευελιξία της C# για την ανάπτυξη γρήγορων και προηγμένων εφαρμογών.
Στη συνέχεια, εισήλθα στον κόσμο της διαδικτυακής ανάπτυξης. Ξεκίνησα να μαθαίνω τα βασικά στοιχεία των ιστοσελίδων, όπως HTML, CSS και JavaScript. 
Αυτή η επέκταση των γνώσεων μου ήταν σημαντική, καθώς με οδήγησε στη δημιουργία του δικού μου ιστότοπου ως ένα προσωπικό project. 
Αυτό το project χρησίμευσε όχι μόνο ως περιοχή δοκιμών, αλλά και ως πρακτική για την εφαρμογή των νέων μου γνώσεων.
Μέσα από αυτή τη διαδικασία, κατάλαβα πώς συνδυάζονται HTML, CSS και JavaScript για τη δημιουργία δυναμικών και αποτελεσματικών ιστοσελίδων.

##Τρίτο Έτος: Συναρπαστική Εξερεύνηση της Java και του Spring Boot:
Στο τρίτο έτος της εκπαίδευσής μου, εκτός από τη συνεχή εξερεύνηση του προγραμματισμού, ειδικεύτηκα περαιτέρω στη Java και τον αντικειμενοστραφή προγραμματισμό (OOP). 
Δημιούργησα μια εφαρμογή επιφανείας εργασίας σε Java, που μπορεί να συγκρίνει αρχεία Excel, προσφέροντας λύσεις για πρακτικές εργασιακές ανάγκες.
Συνεχίζοντας το ταξίδι μου, έπλευσα προς την εξειδίκευση στις διαδικτυακές εφαρμογές, το οποίο είναι μια συναρπαστική περιπέτεια. 
Με χαρά και αφοσίωση, εξερευνώ τον κόσμο της ανάπτυξης διαδικτυακών εφαρμογών, χρησιμοποιώντας το Spring Boot και το Maven κατασκευάζοντας διαδικτυακές εφαρμογές σε Java.
Κατά τη διάρκεια αυτού του ταξιδιού, καλύπτοντας τόσο το front end όσο και το back end με το MVC μοντέλο αρχιτεκτονικής, γνώρισα τα JPA, Thymeleaf, Bootstrap, Spring Security, Lombok και άλλα εργαλεία. 
Το αποτέλεσμα ήταν η δημιουργία δύο εφαρμογών που αντικατοπτρίζουν τη δεξιότητα και την καινοτομία, προσφέροντας λειτουργικά και ευέλικτα διαδικτυακά περιβάλλοντα.
